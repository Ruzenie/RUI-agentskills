# SVG/Canvas 速查（精华版）

## 1. 选型建议
- 常规 UI 图标优先 SVG。
- 高频重绘或复杂动画再考虑 Canvas。

## 2. SVG 基线
- 统一 viewBox
- 统一描边端点与连接策略
- 避免无意义 path 拆分

## 3. Canvas 基线
- 按 DPR 缩放
- 绘制逻辑与事件逻辑分离
- 提供可回放渲染入口

## 执行检查清单
- [ ] 输入前置条件已确认
- [ ] 关键约束已记录
- [ ] 输出格式可被下游技能消费
- [ ] 风险项与回滚思路明确

## 常见误区
- 只给结论，不给依据
- 输出不可执行或无法验证
- 忽略边界条件与异常路径
