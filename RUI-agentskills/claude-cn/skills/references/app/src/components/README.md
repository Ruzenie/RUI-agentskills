# Components Snapshot (CN package)

该目录用于保留 `app/src/components/*` 的本地引用入口，避免技能文档指向包外路径。

## 约定
- 仅作为技能引用锚点。
- 不作为真实业务组件来源。
- 真实项目中应优先读取目标仓库自身的组件目录。

## 执行检查清单
- [ ] 输入前置条件已确认
- [ ] 关键约束已记录
- [ ] 输出格式可被下游技能消费
- [ ] 风险项与回滚思路明确

## 常见误区
- 只给结论，不给依据
- 输出不可执行或无法验证
- 忽略边界条件与异常路径


## 推荐结构
- `ui/`：通用基础组件
- `blocks/`：业务区块组件
- `pages/`：页面组合层

## 迁移提示
- 优先从真实项目组件目录读取
- 仅在缺失时回退本地参考快照
