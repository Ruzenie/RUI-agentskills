# SVG/Canvas 图标技能（精华版）

## 1. 目标
输出一致、可维护、可复用的图标系统资产。

## 2. 输入
- 图标语义清单
- 风格方向
- 平台约束（Web/App/大屏）

## 3. 输出
- `icon.manifest.json`
- 图标源文件（SVG 优先）
- 使用规范说明

## 4. 质量检查
- 视觉一致性
- 对齐与留白一致
- 体积与渲染性能可控

## 执行检查清单
- [ ] 输入前置条件已确认
- [ ] 关键约束已记录
- [ ] 输出格式可被下游技能消费
- [ ] 风险项与回滚思路明确

## 常见误区
- 只给结论，不给依据
- 输出不可执行或无法验证
- 忽略边界条件与异常路径
