# Prompt Patterns

Use these templates when a generation-ready prompt is requested.

## A) UI Redesign Prompt

Redesign a [product type] interface for [target audience].
Style direction: [chosen direction].
Brand adjectives: [adj1], [adj2], [adj3].

Constraints:
- Platform: [web/mobile]
- Visual priority: [primary goal]
- Accessibility baseline: WCAG AA contrast for body text

Design system tokens:
- Typography scale: [define]
- Spacing scale: [define]
- Radius/elevation: [define]
- Color roles: surface/background/text/border/primary/success/warning/error

Screen requirements:
- Include [sections/components]
- Emphasize [primary CTA]
- States required: default/hover/focus/disabled/loading/error

Output:
- Production-ready UI spec with rationale and token table.

## B) Landing Page Prompt

Create a distinctive landing page for [product].
Avoid generic SaaS look. Use [style direction].

Required blocks:
- Hero with sharp value proposition
- Proof section (logos/testimonials/case snippets)
- Feature grid with scannable hierarchy
- CTA section with urgency

Visual guidance:
- Use strong hierarchy before ornament
- Keep palette constrained and semantic
- Ensure mobile layout remains clear and breathable
