# UI Selector Prompt Pack

## Prompt A: Recommendation
Input:
- framework
- project type
- priority matrix
- team constraints

Output:
- top 3 recommendations
- rationale and tradeoffs
- risk and mitigation notes

## Prompt B: Evaluation
Input:
- candidate libraries
- weighted dimensions

Output:
- per-dimension score matrix
- total weighted score
- final ranking and recommendation

## Prompt C: Migration Plan
Input:
- selected target library
- current stack and debt profile

Output:
- phased migration path
- rollback checkpoints
- acceptance criteria per phase
