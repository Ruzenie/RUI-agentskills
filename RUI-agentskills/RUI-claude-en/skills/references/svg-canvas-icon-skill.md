# SVG/Canvas Icon Skill (Essential Version)

## 1. Goal
Output consistent, maintainable, and reusable icon system assets.

## 2. Inputs
- Icon semantic list
- Style direction
- Platform constraints (Web/App/Large Screen)

## 3. Outputs
- `icon.manifest.json`
- Icon source files (SVG preferred)
- Usage guidelines

## 4. Quality Checks
- Visual consistency
- Consistent alignment and whitespace
- Controllable size and rendering performance

## Execution Checklist
- [ ] Input preconditions confirmed
- [ ] Key constraints documented
- [ ] Output format consumable by downstream skills
- [ ] Risk items and rollback approach clarified

## Common Pitfalls
- Providing conclusions without rationale
- Outputs that are not actionable or verifiable
- Ignoring boundary conditions and exception paths
